# CollabRoom

A real-time collaborative coding and whiteboard platform built using MERN, Socket.io, and WebRTC.

## Features

- Real-time room creation and joining
- Live chat
- Shared whiteboard (coming soon)
- Collaborative code editor (coming soon)
- Live cursor tracking (coming soon)
- Video calling with WebRTC (coming soon)

## Tech Stack

Frontend:
- React
- Vite
- React Router
- Socket.io Client

Backend:
- Node.js
- Express.js
- Socket.io

## Getting Started

### Backend

cd server
npm install
npm run dev

### Frontend

cd client
npm install
npm run dev
